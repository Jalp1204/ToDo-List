# 📝 Advanced To-Do List

A feature-rich To-Do List application built using **HTML, CSS, and JavaScript**. This project demonstrates core JavaScript concepts including DOM manipulation, event handling, array methods, filtering, local storage, and dynamic UI updates.

## ✨ Features

- ✅ Add new tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ View Total, Completed, and Pending task counts
- ✅ Filter tasks (All / Pending / Completed)
- ✅ Search tasks instantly
- ✅ Save tasks using Local Storage
- ✅ Press **Enter** to add a new task
- ✅ Responsive and clean UI

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/todo-list.git
```

2. Navigate to the project folder

```bash
cd todo-list
```

3. Open `index.html` in your browser.
